# Welcome!

This folder is the root of the DalFSAE Obsidian vault. 
# Setup

>[!info] Documentation Setup
>This section describes how to setup everything you need to edit the DMS vehicle CAD and documentation. Before attempting this setup, you must install either Git, or the GitHub Desktop App.
## 1. Create a folder named`C:\DalFSAE\`

## 2. Clone [DalFSAE/docs](https://github.com/DalFSAE/docs) into this folder.

```powershell
# Open powershell and run:
cd C:\
mkdir DalFSAE
cd C:\DalFSAE
git clone git@github.com:DalFSAE\docs.git
```
## 2. Run the Setup Script

```
cd C:\DalFSAE\docs
.\setup.ps1
```
## 3. Open /DalFSAE as vault in Obsidian
